# Introduction

Welcome to **AHK v2 Simplified**!

I created this guide because while the official AutoHotkey v2 documentation is comprehensive, it can sometimes be difficult to parse if you just want to get your script working. 

This documentation aims to be:
- **Concise**: We get straight to the point.
- **Practical**: We focus on real-world use cases.
- **Easy to read**: We avoid unnecessary technical jargon where possible.

## Getting Started

In this guide, you will learn the basics of AutoHotkey v2, how to declare variables, write functions, and automate your daily tasks.
